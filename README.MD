# AUTO SCALING ENGINE

A real-time workload processing and auto-scaling simulation platform built using FastAPI, React.js, and Python. The system dynamically scales worker pools based on CPU usage, memory utilization, and incoming request load to simulate cloud-native workload management and distributed processing.

---

## 🚀 FEATURES

- ⚡ Dynamic Auto Scaling
  - Simulates scaling based on CPU, memory, and request load
  - Real-time worker allocation and deallocation
  - Efficient workload balancing

- 🔄 Producer-Consumer Pipeline
  - Concurrent task processing
  - Dynamic worker pool management
  - Queue-based workload distribution

- 📊 Real-Time Monitoring Dashboard
  - Live system metrics visualization
  - CPU and memory tracking
  - Active worker monitoring

- ⚙️ High-Performance Backend
  - RESTful APIs using FastAPI
  - Async task processing
  - Optimized concurrency handling

- 🧩 Scalable Architecture
  - Modular backend structure
  - Extensible scaling policies
  - Fault-tolerant processing pipeline
